# Simple OpenGL viewer

## Status 

Not yet working.

## Requirements 

* C++11 compiler
* OpenGL 2.x
* cmake

## Build

```
$ mkdir build
$ cd build
$ cmake ..
$ make
```

## Run

```
$ ./usdz_glview <input.usdz>
```
